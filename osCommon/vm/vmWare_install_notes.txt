VMware offers lots of products and services.
It can be confusing to find the basic free vm player.
The product you want install is named:

VMware Workstation Player

To download:
    https://customerconnect.vmware.com/en/downloads/info/slug/desktop_end_user_computing/vmware_workstation_player/16_0


*** LINUX:

    To install the .bundle program:
        https://askubuntu.com/questions/521199/how-to-install-bundle-packages-in-ubuntu
        chmod +x file.bundle
        sudo ./file.bundle

    To uninstall:
        https://askubuntu.com/questions/151099/how-to-uninstall-vmware-player
        sudo vmware-installer -u vmware-player
